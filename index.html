Aviso de Privacidad

My Royale Notes respeta la privacidad de sus usuarios.

La aplicación permite crear y gestionar notas personales, así como utilizar funciones asistidas por inteligencia artificial para el procesamiento de texto.

Datos que se pueden procesar:

  • Texto que el usuario escribe o decide analizar voluntariamente.
  • Imágenes que el usuario selecciona de forma manual para funciones específicas (por ejemplo, descripción o lectura de texto).

Uso de la información:

  • Los datos se utilizan únicamente para proporcionar las funciones de la aplicación.
  • No se venden, comparten ni utilizan con fines publicitarios.

Inteligencia artificial:

Algunas funciones utilizan servicios de inteligencia artificial para generar resúmenes o procesar contenido. Estas funciones se ofrecen como apoyo al usuario y no sustituyen el criterio humano.

Almacenamiento:

La información se conserva únicamente el tiempo necesario para ofrecer el servicio solicitado por el usuario.
